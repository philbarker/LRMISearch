LRMI Search

Search Google and then filter the results to see which sites use LRMI properties in schema.org markup.

This search is intended to help find sites using schema.org / LRMI, rather than use LRMI to help find resources.

  • Search for a common word (e.g. 'the') and then use the filters to find pages that contain that word and one or any of the LRMI terms in schema.org.
  • Can be used with a tool such as the OpenLink Structured Data Sniffer (OSDS) to show how the LRMI properties are being used the pages at urls returned.

Notes

The search is brittle in a number of ways, notably the number of results is over-reported. Separate to this some of the results returned do not currently contain the LRMI you might think it does.

Searches for properties in microdata and RDFa only, will not find LRMI being used in JSON-LD.
